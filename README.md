# ロングロール — 最長軌跡チャレンジ

地形のどこかに玉を置いて、転がりを最大化する2Dパズルゲーム（全100ステージ）。

**遊び方:** `index.html` をブラウザで開くだけで動作します（依存なし・単一ファイル）。

## ルール

- マップをクリックすると玉が置かれ、重力で低い方へ転がり、摩擦で必ず停止します。停止までの動きがステージのモードで採点されます。
- **モード（ステージごとに固定・4種を循環）**
  - 🪂 **空気抵抗**: 強い空気抵抗（線形＋速度2乗）の中で道のりの長さを競う。転がり摩擦は通常のまま。
  - 🔷 **凸包サイズ**: 軌跡の凸包面積の平方根 √A。一直線の往復は面積0、広く「面」を囲む動きが評価される。
  - 🎯 **直線距離**: スタートから最も離れた瞬間の直線距離（ループ跨ぎ積算）。
  - 🌀 **慣性半径**: 重心からの広がり Rg（道のり加重の二乗平均平方根）。しきい値パラメータのない純幾何評価。
- **評価:** 理論値の 70% で ★☆☆ ／ 90% で ★★☆ ／ 95% で ★★★
- **解放仕様:** Stage 1〜3 は最初から解放。以降は★の合計が2個増えるごとに次の1ステージが解放。
- **特殊条件**（後半ほど頻出、さらに後半は2つ重なることも）
  - 🐌 抵抗2倍 ／ ⛰ 入山規制（一定標高以上に設置不可・理論値探索も同条件） ／ 🌍 重力2倍
- マップは上下左右がループ（トーラス構造）。地形はシード生成で、平坦なプラトーも出現します。
- ゲーム内「❓ ヘルプ」に全計算式（慣性半径の定義・物理モデル・定数）を記載。

## 理論値について

理論値＝そのステージ・モードで可能な最良スコア。ステージ読み込み時に、プレイヤーと同一の物理シミュレータで開始点を全域グリッド探索＋局所絞り込みして算出します（初回のみ数秒、結果はキャッシュ）。

## 実装メモ

- 地形: 整数波数のサイン波和（トーラス上で完全周期）＋標高帯の平坦化変換 h→f(h)（勾配は f'(h) 倍、C¹連続）。
- 物理: dv/dt = −g∇h − (c₁+c₂|v|)v − μg·v̂。エネルギー単調減少により停止性を保証。
- 進捗は localStorage に保存されます。
