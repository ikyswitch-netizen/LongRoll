# ロングロール — 最長軌跡チャレンジ

地形のどこかに玉を置いて、玉の転がりを最大化する2Dパズルゲーム。全100ステージ。

**遊び方:** `index.html` をブラウザで開くだけで動作します（依存なし・単一ファイル）。

## ルール

- マップをクリックすると玉が置かれ、重力で低い方へ転がり、必ず停止します。何度でも置き直せます（ベストが記録されます）。
- **モード**: ステージごとに4種のモードが順番に割り当てられています（Stage 1=🪂, 2=🔷, 3=📏, 4=🌀, 5=🪂, …）。
  - 🪂 **空気抵抗**: 転がり摩擦は通常のまま、**空気抵抗（線形＋速度2乗）が非常に強い**物理。停止までの**道のりの長さ**がスコア。
  - 🔷 **凸包サイズ**: 軌跡の**凸包面積の平方根 √A** がスコア。一直線の往復は面積0。広く「面」を囲む動きが評価される（破線の多角形で表示）。
  - 📏 **直線距離**: スタート地点から**最も遠くに到達した直線距離**がスコア（トーラスのループ跨ぎは積算。破線の円で表示）。
  - 🌀 **慣性半径**: 軌跡の**重心からの広がり（慣性半径 Rg・道のり加重）**がスコア。しきい値パラメータのない純幾何評価（重心とRg円を破線で表示）。
- **評価**: 理論値の 70% で ★☆☆ ／ 90% で ★★☆ ／ 95% で ★★★
- **アンロック**: Stage 1〜3 は最初から解放。**★の合計が2個増えるごと**に次のステージが1つ解放されます（Stage n の解放条件は合計★ 2×(n−3) 個）。
- **特殊条件**（後半のステージほど頻出、さらに後半は2つ重なることも。カードにアイコン表示）:
  - 🐌 **抵抗2倍**（摩擦・空気抵抗×2） ／ ⛰ **入山規制**（一定標高以上の赤いエリアに設置不可。理論値探索も同条件） ／ 🌍 **重力2倍**
  - 🌋 **地殻変動**（青破線より低地に玉が入ると、地形の高低が約3秒かけて h→−h に反転する。1回だけ。反転中は停止しない）
  - 🚀 **ジェットスタート**（転がり出した瞬間、最急降下方向へ初速15。平坦地に置くと不発）
  - ⛏ **トンネル**（進行方向の上り勾配0.15超は勾配をクランプ＋横方向の力を無視して直進＝掘り進む。軌跡はオレンジ表示。地形データは不変なのでリトライで元通り）
- マップは上下左右がループ（トーラス構造）。各ステージはシード値から決定論的に生成され、後半ほど地形が複雑になります。**平坦なプラトー**（勾配0の帯）も出現し、玉の停止地形として機能します。

## 理論値について

理論値＝そのステージで可能な最大スコア。ステージ読み込み時に、プレイヤーと同一の物理シミュレータで開始点を全域グリッド探索＋局所絞り込みして算出します（初回のみ数秒、結果はキャッシュ）。

## 計算式

共通の運動方程式（G=25）:

```
a = −G∇h(p) − (c₁ + c₂|v|)·v − μG·v/|v|
```

- 通常物理（🔷凸包サイズ・📏直線距離・🌀慣性半径）: c₁=0.1, c₂=0, μ=0.05
- 🪂空気抵抗モード: c₁=0.2, c₂=0.05, μ=0.05（摩擦は通常のまま、空気抵抗だけが大幅に強い）
- 特殊条件: 🐌抵抗2倍で μ, c₁, c₂ が2倍 ／ 🌍重力2倍で G が2倍

スコア:

- 🪂 軌跡長: `L = ∫|v|dt`
- 🔷 凸包サイズ: `√A`（A＝約1間隔でサンプリングした軌跡点の凸包の面積）
- 📏 直線距離: `max_t |p(t) − p(0)|`（非ラップ座標）
- 🌀 慣性半径: `p̄ = (1/L)∫p ds`, `Rg = √((1/L)∫|p−p̄|²ds)`（ds＝道のり要素）

詳しい解説と、慣性半径という評価にたどり着くまでの試行錯誤の裏話は、ゲーム内ヘルプ（「📐 スコアの計算式」「🗒 裏話」）に掲載しています。

## 実装メモ

- 地形: 整数波数のサイン波和（トーラス上で完全周期）。ステージが進むほど波の本数・空間周波数が増加。
- 摩擦によりエネルギーが単調減少するため停止性を保証（道のり ≦ 落差/μ）。
- 進捗は localStorage に保存されます。
