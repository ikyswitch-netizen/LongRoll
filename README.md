# ロングロール — 最長軌跡チャレンジ

地形のどこかに玉を置いて、玉の転がりを最大化する2Dパズルゲーム。全100ステージ。

**遊び方:** `index.html` をブラウザで開くだけで動作します（依存なし・単一ファイル）。

## ルール

- マップをクリックすると玉が置かれ、重力で低い方へ転がり、必ず停止します。何度でも置き直せます（ベストが記録されます）。
- **モード**: ステージごとに3種のモードが順番に割り当てられています（Stage 1=🪂, 2=📏, 3=🌀, 4=🪂, …）。
  - 🪂 **空気抵抗**: 転がり摩擦がほぼなく、空気抵抗（速度依存）で減速する滑走物理。停止までの**道のりの長さ**がスコア。玉は非常に長く走ります。
  - 📏 **直線距離**: スタート地点から**最も遠くに到達した直線距離**がスコア（トーラスのループ跨ぎは積算。破線の円で表示）。
  - 🌀 **慣性半径**: 軌跡の**重心からの広がり（慣性半径 Rg・道のり加重）**がスコア。しきい値パラメータのない純幾何評価（重心とRg円を破線で表示）。
- **評価**: 理論値の 70% で ★☆☆ ／ 90% で ★★☆ ／ 95% で ★★★
- **アンロック**: Stage 1〜3 は最初から解放。**★の合計が2個増えるごと**に次のステージが1つ解放されます（Stage n の解放条件は合計★ 2×(n−3) 個）。
- マップは上下左右がループ（トーラス構造）。各ステージはシード値から決定論的に生成され、後半ほど地形が複雑になります。

## 理論値について

理論値＝そのステージで可能な最大スコア。ステージ読み込み時に、プレイヤーと同一の物理シミュレータで開始点を全域グリッド探索＋局所絞り込みして算出します（初回のみ数秒、結果はキャッシュ）。

## 計算式

共通の運動方程式（G=25）:

```
a = −G∇h(p) − (c₁ + c₂|v|)·v − μG·v/|v|
```

- 通常物理（📏直線距離・🌀慣性半径）: c₁=0.1, c₂=0, μ=0.08（転がり摩擦が無視できない強さ、約10秒で停止）
- 🪂空気抵抗モード: c₁=0.045, c₂=0.01, μ=0.01（速度2乗の空気抵抗が主体で、通常より長く滑走）

スコア:

- 🪂 軌跡長: `L = ∫|v|dt`
- 📏 直線距離: `max_t |p(t) − p(0)|`（非ラップ座標）
- 🌀 慣性半径: `p̄ = (1/L)∫p ds`, `Rg = √((1/L)∫|p−p̄|²ds)`（ds＝道のり要素）

詳しい解説と、慣性半径という評価にたどり着くまでの試行錯誤の裏話は、ゲーム内ヘルプ（「📐 スコアの計算式」「🗒 裏話」）に掲載しています。

## 実装メモ

- 地形: 整数波数のサイン波和（トーラス上で完全周期）。ステージが進むほど波の本数・空間周波数が増加。
- 摩擦によりエネルギーが単調減少するため停止性を保証（道のり ≦ 落差/μ）。
- 進捗は localStorage に保存されます。
