# ロングロール — 最長軌跡チャレンジ

地形のどこかに玉を置いて、転がる軌跡の長さを最長にする2Dパズルゲーム。

**遊び方:** `index.html` をブラウザで開くだけで動作します（依存なし・単一ファイル）。

## ルール

- マップをクリックすると玉が置かれ、重力で低い方へ転がり、摩擦で必ず停止します。
- **ゲームモード**（進捗・理論値はモード別に記録）
  - 🧵 **軌跡モード**: 停止するまでの道のりの合計の長さがスコア。
  - 🚀 **飛距離モード**: スタート地点から最も遠くに到達した直線距離がスコア（トーラスのループ跨ぎは積算）。高低差よりも「どの方向へ・どの谷筋を走らせるか」が問われます。
- 何度でも置き直せます（ベストが記録されます）。
- **評価:** 理論値の 70% で ★☆☆ ／ 90% で ★★☆ ／ 95% で ★★★
- **アンロック仕様:** ステージ n をクリア（達成率 70% 以上 ＝ ★1）すると、ステージ n+1 と n+2 の2つがアンロックされます。
- マップは上下左右がループ（トーラス構造）。
- 各ステージはシード値から決定論的に生成され、後半ほど地形が複雑になります（全30ステージ）。
- 高さ表示は「カラー」「等高線」を切り替え可能。

## 理論値について

理論値＝そのステージで可能な最長軌跡。ステージ読み込み時に、プレイヤーと同一の物理シミュレータで開始点を全域グリッド探索＋局所絞り込みして算出します（初回のみ数秒、結果はキャッシュ）。

## 実装メモ

- 地形: 整数波数のサイン波和（トーラス上で完全周期）。ステージが進むほど波の本数・空間周波数が増加。
- 物理: 重力 (-G∇h) ＋ 粘性抵抗 ＋ クーロン転がり摩擦。エネルギー単調減少により停止性を保証。
- 進捗は localStorage に保存されます。
